# SQL-Driven Sales Analytics

## Project Objective
This project demonstrates a SQL-first approach to sales data analysis and business reporting using Power BI.

## Tools Used
- SQL (Joins, Aggregations, Window Functions)
- Power BI (DAX, Data Modeling, Dashboards)

## Project Workflow
1. Sales data loaded and cleaned using SQL
2. Business logic implemented through SQL queries
3. Power BI used as a visualization and reporting layer

## Key Analysis
- Region-wise sales performance
- Monthly sales trends
- Top products by revenue
- Profit margin analysis

## Key Insights
- Certain regions generate high sales but lower profit margins
- Top products contribute a major portion of total sales
- Seasonal trends observed in monthly sales
